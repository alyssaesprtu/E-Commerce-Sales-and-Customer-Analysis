# E-Commerce-Sales-and-Customer-Analytics
An end-to-end e-commerce analytics project using SQL, Python, and Power BI to analyze sales performance, customer behavior, seller efficiency, and return trends.

## Objectives:
1. Analyze sales and revenue trends
2. Identify high-performing products and sellers
3. Understand customer purchasing behavior
4. Measure return and payment patterns
5. Build interactive dashboards for business insights

## Datasets
- customers
- orders
- order_items
- payments
- products
- returns
- sellers

source: https://www.kaggle.com/datasets/josephbenofficial/shopin?resource=download

## Tools and Technologies
- SQL (MySQL)
- Python
- Pandas
- Matplotlib / Seaborn
- Power BI
- Jupyter Notebook
